# Book of Tales Examples

These are example Book of Tales compatible with the [Book of Inifite Tales](https://book-of-infinite-tales.com) for use with the board game *Tales of the Arthurian Knights*.

To use this example, go to the Book of Infinite Tales website and then enter this location and choose an example book of tales.

```
RobMcA/book-of-tales-example
```
